# Ethereum Price Checker
#### By Ash Laidlaw

## About

This is a simple, JavaScript web app that will check what the current price of Ethereum is. It was created to allow for a quick and simple price check for Ethereum.

## Technologies Used

* HTML
* JavaScript
* SASS
* Cryptocompare API
